# Data Engineering Roadmap Repository

이 레포지토리는 데이터 엔지니어 취업 준비와 학습 기록을 위해 구성되었습니다.  
SQL 고급, ETL, 데이터베이스, 분산 처리, 클라우드, 인프라, BI까지 신입 데이터 엔지니어가 다루는 내용들을 정리합니다.  
각 폴더는 실습 코드와 학습 내용을 포함하며, 일부 프로젝트는 포트폴리오로 활용합니다.  

---

## Repository Structure

01_sql_etl_basics/     → SQL 고급 문법 & ETL 기본기  
02_data_ingestion/     → 데이터 수집 및 로딩 (CSV, API, Kafka 등)  
03_sql_database/       → 관계형 DB, NoSQL, DWH 설계 및 최적화  
04_data_engineering/   → Spark, Airflow, Kafka 등 데이터 엔지니어링 핵심 툴  
05_cloud/              → AWS/GCP를 활용한 클라우드 파이프라인  
06_infra_ops/          → Docker, Kubernetes, Terraform, Jenkins, 모니터링  
07_visualization_bi/   → Tableau, Power BI, Metabase 등 BI 툴  
08_projects/           → 미니/중급 데이터 엔지니어링 프로젝트 모음  
09_docs/               → 면접 대비 Q&A, 용어 정리, 로드맵  

---
 
- 학습 기록은 블로그(Tistory 등)에 상세히 정리했고, 레포에는 **실습 코드와 프로젝트 중심**으로 관리합니다.  
- 프로젝트(`08_projects/`)는 취업 시 이력서/포트폴리오에 직접 링크할 수 있습니다.
