# 01. SQL & ETL Basics

이 폴더는 SQL 심화와 ETL 기초를 다룹니다. 데이터 엔지니어로서 기본기를 탄탄히 하기 위한 단계입니다.

## 📌 학습 내용
- SQL 고급 문법: Window Function, CTE, Subquery, Indexing
- ETL 기초: CSV → DB 적재, Pandas Transform, 간단한 파이프라인
- 데이터 모델링: ERD 설계, 정규화/비정규화
- dbt 기초

## 🛠️ 폴더 구조
- `sql_advanced/` : 고급 SQL 실습
- `etl_foundation/` : Pandas, DB 적재
- `data_modeling/` : ERD, 스키마 설계
- `dbt_intro/` : dbt 실습 코드

## 🎯 학습 목표
- SQL로 복잡한 데이터 처리 능력 확보
- ETL 파이프라인 기본 구조 이해
- 데이터 모델링 감각 형성
